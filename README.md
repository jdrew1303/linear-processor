# linear-processor
Concept DIY linear processor for 16mm and Super8 film.

![Tank concept](/img/linear-processor-tank-1.png?raw=true "Tank concept")

###Materials
- Arduino
- 15RPM motor
- 1/4" acrylic (lots)
- ABS filament (PLA will absorb water if untreated, not ideal)
- Acetone

###Tools
- Laser cutter or CNC
- 3D printer
- Sandpaper
